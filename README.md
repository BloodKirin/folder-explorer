扫描目录，分析文件结构和统计信息，给任意文件添加备注，导出带注释的树形文本和其它多种数据格式。

![](https://raw.githubusercontent.com/d2-projects/folder-explorer/master/docs/image/banner/app-preview.png)

## 功能

### 分析

* [x] 文件结构
* [x] 文件种类统计
* [x] 文件容量分布
* [ ] 代码行数统计

### 导出功能

* [x] 带注释的树形文本
* [x] json
* [x] xmind 脑图
* [x] XML
* [x] HTML
* [ ] 表格
